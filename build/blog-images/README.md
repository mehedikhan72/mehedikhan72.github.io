## Adding Images

To add images:
1. Add your image files to this directory
2. Reference them in blog markdown with: `![Alt text](/blog-images/filename.jpg)`
3. Optimize images for web (recommended max width: 1200px)

## Image Guidelines

- Format: JPG or PNG
- Max size: 500KB per image
- Dimensions: 1200x800px recommended
- Compression: Use tools like TinyPNG or ImageOptim
